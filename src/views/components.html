Customize the css

  Customize the required css class

You can override the css class .fl-frm__lbl--required:before
The required styles are part of the floating label pattern but still you can change the default ones and add your own styles.

  Customize the required css class

You can override the css class .fl-frm__tooltip
If you are have your own tooltips you can still use them the way you want and you can also modify the default ones like changing font-size or colors maintaing your own styles.
