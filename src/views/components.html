Angular floating labels are custom directives

Add always fl-label attribute to the input element, you don't need to use thelabelelement.
For the input elements and textareas, the placeholder attribute is going to act like a label and for the select elements the first option is going to act like a label.

  Customize the required css class

You can override the css class .fl-frm__lbl--required:before
The required styles are part of the floating label pattern but still you can change the default ones and add your own styles.
